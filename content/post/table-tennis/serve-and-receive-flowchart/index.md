這篇是把前一篇的發球與接發球決策樹，整理成更像流程圖的版本。

適合用途：

- 比賽前快速複習
- 練球前先跑一次腦內流程
- 幫自己建立固定的前三板判斷順序

---

# 一、發球流程圖

```text
開始發球前
  ↓
先問自己：我想要對方回什麼球？
  ↓
我這一分想打哪種節奏？
  ├─ 想穩定先拿主動
  │    → 發短下旋 / 短不轉 到反手或中間
  │    → 預設第三板壓反手 / 中間
  │
  ├─ 想逼對方先判斷失誤
  │    → 發短不轉 / 偽下旋
  │    → 等對方冒球後主動
  │
  ├─ 想打亂對方站位
  │    → 偶爾發長球到反手 / 追身
  │    → 預設下一板先壓中間或反手
  │
  └─ 對方擰拉能力強
       → 短球更短、更低、旋轉更明確
       → 不給舒服半出台機會
```

---

# 二、第三板流程圖

```text
對方回球後
  ↓
先判斷回球品質
  ├─ 高、慢、落點舒服
  │    → 主動加速
  │    → 有把握可側身
  │
  ├─ 普通，但我可先上手
  │    → 先壓反手 / 中間
  │    → 不急著一板打死
  │
  ├─ 短、低、品質不差
  │    → 先穩處理
  │    → 把目標放到第四板
  │
  └─ 比預期更強
       → 別硬打
       → 先活進相持
```

---

# 三、側身第三板流程圖

```text
這球要不要側身？
  ↓
符合條件幾個？
  - 球夠高
  - 球夠慢
  - 落點夠中間 / 偏正手
  - 側身後回得來中間
  ↓
  ├─ 符合 3 個以上
  │    → 可側身
  │
  └─ 不足 3 個
       → 不勉強側身
       → 先做穩定第三板拿主動
```

---

# 四、接發球總流程圖

```text
對方發球來了
  ↓
先判斷長度
  ├─ 半出台 / 稍微出台
  │    → 優先思考主動起板
  │
  ├─ 很短
  │    → 優先思考擺短 or 挑 / 擰
  │
  └─ 明顯長球
       → 優先進攻或帶起來
```

接著：

```text
再判斷旋轉 + 自己站位
  ↓
  ├─ 看得清、站得到、來得及
  │    → 主動優先
  │
  ├─ 很短、很低、很轉，自己不舒服
  │    → 高品質短控制優先
  │
  └─ 旋轉重、判斷不夠清楚，但可送長有目的
       → 長球控制也可以是正解
```

---

# 五、半出台球流程圖

```text
如果球是半出台
  ↓
先看旋轉
  ├─ 下旋明確，但不是特別重
  │    → 擰 / 帶 / 拉起來
  │
  ├─ 不轉或弱下旋
  │    → 更應該主動處理
  │
  ├─ 自己看不清、腳沒到位
  │    → 不硬起
  │    → 先控制回去也可以
  │
  └─ 起板能上，但質量很普通
       → 再想一遍：
         這球起板真的比控制更賺嗎？
```

---

# 六、短球流程圖

```text
如果球很短
  ↓
先看：可不可以高質量主動？
  ├─ 可以（不轉 / 弱下旋 / 可進球）
  │    → 擰 / 挑 / 帶
  │
  ├─ 不太可以（很短、很低、很轉）
  │    → 高品質擺短
  │
  └─ 對方就在等你先上手反打
       → 擺短常常比硬擰更賺
```

---

# 七、搓長流程圖

```text
如果要搓長
  ↓
先問：我有沒有目的？
  ├─ 深反手
  ├─ 深中間
  ├─ 追身
  ├─ 重下旋
  └─ 改節奏
  ↓
  ├─ 有明確目的
  │    → 這是戰術，不是退讓
  │
  └─ 沒有目的
       → 那只是把球送回去
```

---

# 八、最終總流程

```text
發球：
先決定第三板 → 再決定發什麼

接發：
先判斷長度 → 再判斷旋轉與站位 →
能高質量主動就主動，
不能高質量主動就高質量控制
```

---

# 九、超短口訣版

## 發球
- 先想第三板
- 先讓對方不好回
- 第三板先主動，不一定先打死

## 接發
- 看得清、站得到，就主動
- 太短太轉，不硬擰，先高品質控制
- 要送長，就送深、送刁、送有目的

---

# 十、最後一句話

# **流程圖的目的不是讓自己變僵，而是讓關鍵時刻少亂。**

真正成熟的前三板，不是每球都最兇，而是每球都更有邏輯。
