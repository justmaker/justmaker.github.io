# 我的桌球技術缺陷追蹤記錄

這篇文章是一份**持續更新**的個人技術缺陷追蹤表。

目的不是寫一次性的檢討報告，而是建立一個可以長期對照的記錄：

- 每次比賽或影片分析後，記錄發現的問題
- 追蹤哪些問題在改善、哪些還在
- 讓訓練方向有依據，不是每次重新猜

---

## 怎麼讀這份記錄

每一次分析會以**日期**為單位，記錄：

- **分析來源**：哪支影片 / 哪場比賽
- **發現的問題**：用問題代碼標記
- **失分原因統計**：如果有做的話
- **狀態標記**：
  - 🔴 **未改善**：問題依然存在
  - 🟡 **改善中**：有意識到，開始調整，但還不穩定
  - 🟢 **已改善**：明顯進步，不再是主要問題
  - ⚪ **新發現**：這次才第一次注意到

## 2026-07-15｜比賽影片分析（橘衣 vs 黑衣）

### 分析來源
比賽影片，兩支合計約 22 分鐘。抽樣 96 張關鍵影格 + 3:15 後失分型態判讀。

### 本次觸發的問題代碼

| 代碼 | 嚴重程度 | 備註 |
|------|---------|------|
| F01 | ⭐⭐⭐ | 所有問題的地基，改了立刻有感 |
| FH01 | ⭐⭐⭐ | 正手力量上限被鎖住的主因 |
| F03 | ⭐⭐⭐ | 被連續調動的根本原因 |
| M01 | ⭐⭐⭐ | 3:15 後失分第一名 |
| M02 | ⭐⭐☆ | 跟 M01 常一起出現 |
| T01 | ⭐⭐☆ | 次要但明顯 |
| BH01 | ⭐☆☆ | 不是最大洞但影響反手反應速度 |
| BH02 | ⭐☆☆ | 限制反手主動性 |
| FH02 | ⭐☆☆ | 正手空間不足 |
| FH03 | ⭐☆☆ | 正手微調不夠 |

### 3:15 後失分原因類型統計（估計）

| 失分原因類型 | 對應代碼 | 估計佔比 | 信心 |
|-------------|---------|---------|------|
| 正手大角後復位不及 | M01, F03 | 35%～40% | 高 |
| 正手移動中倉促出手 / 伸手打 | M02, FH01 | 25%～30% | 中高 |
| 中間位被擠住 | T01 | 15%～20% | 中 |
| 反手偏被動 / 偏晚 | BH01, BH02 | 10%～15% | 中 |
| 接發或前兩板問題 | — | 0%～10% | 低 |

### 3:15 後失分時間軸（估計版）

> 說明：這不是逐分逐球的人工記錄，而是依據 3:15 後到影片結尾的畫面抽樣與回合型態，整理出的**估計時間軸**。目的不是假裝精確到每一球，而是把重複出現的失分模式對到具體時間區段，方便之後回看原片。

| 時間點（約） | 對應代碼 | 失分型態 | 判讀重點 |
|-------------|---------|---------|---------|
| 3:18 | M01, F03 | 正手大角後復位不及 | 先被拉到正手大角，第一拍還能碰到，但打完人還在外面，下一板空檔被利用 |
| 3:22 | M02, FH01 | 正手位移動中倉促出手 | 腳沒先到位，身體外傾，出手像在救球，回球品質不足 |
| 3:27 | T01 | 中間位被擠住 | 球進入正反手 crossover 區，出手偏擠、偏晚，節奏被對手接管 |
| 3:31 | M01, F03 | 正手大角後復位不及 | 回合型態再次出現：先被拉開，再因回中不夠快失去下一板位置 |
| 3:36 | BH01, BH02 | 反手偏被動 / 偏晚 | 反手處理較像先擋住，沒有把球頂回去，對手持續壓制 |
| 3:41 | M02, FH01 | 正手位移動中倉促出手 | 手先伸出去夠球，擊球點不理想，像是移動中硬處理 |
| 3:46 | M01, F03 | 正手大角後復位不及 | 第一拍勉強救到，但下一拍回中不及，還是輸在位置 |
| 3:52 | T01 | 中間位被擠住 | 中間位銜接不夠乾淨，正反手切換不順，失去主動 |
| 3:58 | M01, F03 | 正手大角後復位不及 | 後段最典型的失分模式之一：先被拉開，然後回不來 |

### 時間軸版的重點結論

把時間軸攤開來看，會更清楚一件事：

**3:15 後的主要問題不是某一板突然失手，而是「被調動後的後續處理」反覆出問題。**

最頻繁觸發的代碼：
- **M01**（正手大角後復位不及）：4 次
- **M02**（腳沒先到，手先救）：2 次
- **T01**（中間位被擠住）：2 次
- **BH01/BH02**（反手偏被動）：1 次

### 建議優先訓練方向

按問題代碼對應訓練：

| 目標代碼 | 訓練內容 |
|---------|---------|
| F01, F02 | 徒手低重心持拍練習：雙腳略寬於肩，膝蓋彎 120°，維持 30 秒 × 10 組 |
| F03, M01 | 影子步伐：每打一個想像球後立刻 2 步碎步回中，反覆 3 分鐘 |
| F03, M01 | 三點跑位多球：正手位→中路→反手位循環，每次觸球後必須回中 |
| FH01 | 無球空揮：手插腰轉體，感受力量從腰啟動 |
| FH01 | 近台對拉慢速版：專注「先轉腰再出手」 |
| M02 | 左右擺速多球：強迫低姿態橫移，腳先到再出手，每組 20 顆 |
| T01 | 中間位步伐練習：針對 crossover 區快速判斷正反手切換 |

口訣：**「打完就動，不打也動」**

### 7/4 vs 7/15 比對結論

**所有在 7/15 被標記為 🔴 的問題，在 7/4 的四支影片中全部都有出現。**

這代表：

1. **這些不是偶發問題，而是結構性缺陷**
2. **至少從 7/4 到 7/15（11 天），這些問題沒有明顯改善**

4. **M01 和 M02（復位不及、手先腳後）在兩個時間點都是主要失分原因**

也就是說：

**現在看到的問題不是最近才冒出來的。它們早就存在，只是之前沒有系統化記錄。**

### 有沒有新問題需要新增代碼？
