## 學習流程與技術點

我認為學鋼琴的流程和技術點如下：

### 1. 手型、指法、手順、手腕、手臂、身形

這是一切的基礎：

- **和弦**
- **音階**：級進
- **琶音**：跳進
- **輪指**
- **重點**：放鬆、避免受傷、獨立性、訓練強度
- **好看**：起手、收手、移動、跳躍

### 2. 踏板

踏板的使用時機和深淺控制。

### 3. 視譜分析

看懂樂譜的結構、和弦、走向。

### 4. 視奏

邊看譜邊彈的即時反應能力。

## 輪指技巧

1. 指尖觸鍵
2. 按一半深度
3. 向掌心方向收
4. 手腕微抬

## 技術核心八要點

1. **手臂放鬆** — 一切的前提
2. **手型支撐** — 穩定的拱形結構
3. **手指獨立** — 每根手指單獨控制
4. **雙手配合** — 左右手的協調
5. **激活三關節** — 指尖、指根、手腕
6. **基本觸鍵** — 力度與音色控制
7. **前臂的運用** — 大跳和力度變化
8. **指尖的運用** — 精細的音色表現

## 彈奏範圍建議

- **Bass / Chord / Melody 範圍**：倍低音 A 到最高音 C
- **伴奏範圍**：最佳為低音 F 到中音 E；次佳為低音 C 到中音 G
- **主奏範圍**：最佳為中音 G 到倍高音 E
  - 男聲：低音 C 到中音 E
  - 女聲：低音 G 到高音 C
  - 主旋律會和伴奏打架，因此最好移高八度
- **高音盡量加和弦**
